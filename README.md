## 🎵 **About N-Xade Music**

N-Xade Music is an open-source music player application with a modern and user-friendly interface. It has been developed with a focus on **privacy**, **no ads**, and **completely free** experience.

### 🔒 **Privacy First**
- **No data collection** - Your music stays on your device
- **No tracking** - We don't monitor your listening habits
- **No cloud sync** - All data remains local
- **Open source** - Transparent code for everyone to review

[📖 **Read Our Privacy Policy**](PRIVACY_POLICY.md)

| Feature | Description |

## 🚀 **Quick Start**

### **For Users**
1. **Download from F-Droid** (Recommended)
2. **Grant storage permission** for music files
3. **Start listening** to your music!

### **For Developers**
```bash
# Clone the repository
git clone https://github.com/NothingnessN/n-xade-music.git

# Navigate to project
cd n-xade-music

# Install dependencies
flutter pub get

# Run the app
flutter run
```
